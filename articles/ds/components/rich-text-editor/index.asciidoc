== Toolbar actions

=== History

|===
|Button |Title |Description

|
|Undo
|Reverses the previous action

|
|Redo
|Restores actions undone by Undo

|===

=== Emphasis

|===
|Button |Title |Description

|
|Bold
|*Boldens text*

|
|Italic
|_Italicizes text_

|
|Underline
|[.underline]#Underlines text#

|
|Strikethrough
|[.line-through]#Strikethroughs text#

|===

=== Headings

There are three different headings; H1, H2 and H3. Use them in order to signify structure (and importance).

|===
|Button |Title |Description

|
|H1
|

= Heading level 1

|
|H2
|

== Heading level 2

|
|H3
|

=== Heading level 3

|===

=== Subscript & Superscript

|===
|Button |Title |Description

|
|Subscript
|Subscript text is positioned ~below~ the normal baseline and with smaller font size

|
|Superscript
|Superscript text is positioned ^above^ the normal baseline and with smaller font size

|===

=== List

|===
|Button |Title |Description

|
|Ordered list
| .Creates a numbered list

|
|Unordered list
| *Creates a bulleted list

|===

=== Alignment

|===
|Button |Title |Description

|
|Left align
|Left aligns text (default)

|
|Center align
|Center aligns text

|
|Right align
|Right aligns text
|===

=== Image & Link

|===
|Button |Title |Description

|
|Image
|Uploads and inserts an image from your device

|
|Link
|https://vaadin.com[Creates a hyperlink]
|===

=== Blocks

|===
|Button |Title |Description

|
|Block quote
|Creates a section quoted from another source

|
|Code block
|Creates a block of formatted as code
|===

=== Clear

|===
|Button |Title |Description

|
|Clear formatting
|Removes any and all formatting of the selected text
